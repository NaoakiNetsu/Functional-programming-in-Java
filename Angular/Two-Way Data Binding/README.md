# 双方向データバインディング

## 双方向データバインディングって何？

Angular が持つ特徴的な機能の1つで、Javascript のオブジェクト（Model）が保持するデータと、HTMLテンプレート（view）として画面に表示される内容を自動的に連携する機能である。

簡単に言えば、「画面での入力はJavascript上の変数に自動的に反映される」、さらに「Javascript上の変数が変化すると自動的に画面に反映される」といった動きをするので『**双方向**』なのである。
