### クライアントとサーバの通信について理解するために必要な知識をまとめる

#### API
「Application Programming Interface（アプリケーション・プログラミング・インターフェイス）」の略語で、アプリやソフトの機能の使い方を定義したもの。

##### Web API
HTTPを通じてアプリケーションを扱うためのインターフェース。色々公開されている。
例：Twitter API, FaceBook API
